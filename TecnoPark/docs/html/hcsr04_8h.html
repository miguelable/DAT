Detailed Description

Library for managing the HC-SR04 ultrasonic sensor.

Version
0.1
Date

This file contains the declaration of the HCSR04 class, which provides functions to manage the HC-SR04 ultrasonic sensor. The class includes functions to initialize the sensor, start and stop measurements, and retrieve distance and duration data. The class also supports setting a callback function to trigger an action when the distance exceeds a specified threshold.
